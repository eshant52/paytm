# Basic PayTM web application

## Configuration (required)

Add .env file inside backend folder

```dotenv
  MONGO_URI = 
  PORT = 3009
  JWT_SECRETE = 
  BCRYPT_SALT_ROUND = 10
```

## Running steps

1. Clone this repo to run locally.
1. Run following steps

Backend

```bash
cd backend
npm install
npm run start
```

Frontend

```bash
cd frontend
npm install
npm run dev
```
